Explainer - a simple program for explaining things

Why would you use it

To generate some quick documenation about how something works. There are other more complicated systems out there, asciidoctor looked cool, but also complicated.

How it works

You create an input file (input.edn), then execute the 'explainer' program. It walks through each block of the input file and produces output for each section.

For plantUML sections, the plantUML api is invoked and the diagram is generated.

[ 
      :html "<h3>Authentication Examplep<h;/h3>"
      :html-hiccup [:div "Alice authenticates via Bob." [:br] [:br] ] ]
      :plantuml-file "diagram.pu"
]
This input generates the following output;

Authentication Example

Alice authenticates via Bob.


Example usage from inside project

$ git clone https://github.com/bherrmann7/explainer.git 
$ clj -X:run-x && google-chrome docs/index.html # repeat as needed

Install as local command

$ ./install.sh # installs to $HOME/bin/explainer 
$ mkdir ~/cow && cd ~/cow 
$ echo ' [ :html "<h1>Goose</h1>" ] ' > input.edn 
$ explainer && google-chrome docs/index.html
